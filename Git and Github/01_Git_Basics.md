Git is a **distributed version control system** that tracks changes in source code, supports team collaboration, and maintains complete project history.

### Key Features
- **Popular**: Industry standard for version control
- **Free & Open Source**: Publicly available source code that anyone can view, modify, and distribute
- **Fast & Scalable**: Handles projects of any size efficiently
- **Distributed**: Every developer has a complete copy of the project history
