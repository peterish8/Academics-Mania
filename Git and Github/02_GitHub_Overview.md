# GitHub Overview



## üåê What is GitHub?

GitHub is a **web-based platform** that allows developers to store and manage their code using Git. It provides a cloud-based interface for Git repositories with additional collaboration features.

### Key Benefits
- **Cloud Storage**: Your code is safely stored online
- **Portfolio**: Showcase your work to potential employers
- **Collaboration**: Work with teams on shared projects
- **Open Source**: Contribute to public projects worldwide


## üîó GitHub vs Git

| Aspect       | Git                      | GitHub                    |
| ------------ | ------------------------ | ------------------------- |
| **Type**     | Version control software | Web-based hosting service |
| **Location** | Local computer           | Cloud/Internet            |
| **Purpose**  | Track changes            | Store & collaborate       |
| **Access**   | Command line/GUI         | Web browser               |
| **Cost**     | Free                     | Free (with paid plans)    |

## üí° Common GitHub Features

### Repositories
- Store your project files and history
- Public (visible to everyone) or Private

### Issues
- Track bugs and feature requests
- Organize project tasks

### Pull Requests
- Propose changes to code
- Code review process

### Actions
- Automate testing and deployment
- CI/CD workflows
