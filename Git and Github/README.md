## üõ†Ô∏è Essential Commands Quick Reference

```bash
# Setup
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# Basic Workflow
git init                    # Initialize repository
git clone <url>            # Clone repository
git add .                  # Stage all changes
git commit -m "message"    # Commit changes
git push                   # Upload to remote
git pull                   # Download from remote

# Branching
git branch                 # List branches
git checkout -b <branch>   # Create and switch branch
git merge <branch>         # Merge branch

# History
git log --oneline          # View commit history
git status                 # Check current status
```

## üéØ Best Practices Summary

### Commits
- **Atomic**: One logical change per commit
- **Clear messages**: Describe what and why
- **Frequent**: Commit often, push regularly

### Branches
- **Descriptive names**: `feature/login`, `fix/bug-123`
- **Short-lived**: Merge quickly
- **Updated**: Keep in sync with main branch

### Collaboration
- **Pull first**: Always get latest changes
- **Code review**: Use Pull Requests
- **Communication**: Discuss major changes
