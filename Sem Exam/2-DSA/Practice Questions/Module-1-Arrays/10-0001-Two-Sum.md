## ğŸ¯ **Problem Core:**
Find indices of two numbers that add up to target.

## ğŸ’¡ **Key Insight:**
HashMap stores complements. Check if complement exists before adding.

## ğŸ”§ **Optimal Approach:**
**Algorithm:** HashMap

**Steps:**
- HashMap {value: index}
- For each num: complement = target - num
- If complement in map, return indices
- Else add num to map

## ğŸ’» **Code (Run Directly):**
```python
def twoSum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return []

# TEST
nums = [2, 7, 11, 15]
target = 9
print("nums:", nums, "target:", target)
print("Output:", twoSum(nums, target))
```
