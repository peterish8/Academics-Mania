Find floor of square root of x (without using sqrt).

## ğŸ’¡ **Key Insight:**
Binary search for number whose square is <= x.

## ğŸ”§ **Optimal Approach:**
**Algorithm:** Binary Search

**Steps:**
- Binary search from 1 to x
- If mid*mid == x, return mid
- If mid*mid < x, save mid as candidate, search right
- If mid*mid > x, search left

## ğŸ’» **Code (Run Directly):**
```python
def mySqrt(x):
    if x < 2:
        return x
    left, right = 1, x // 2
    while left <= right:
        mid = (left + right) // 2
        square = mid * mid
        if square == x:
            return mid
        elif square < x:
            left = mid + 1
        else:
            right = mid - 1
    return right

# TEST
print("sqrt(4):", mySqrt(4))
print("sqrt(8):", mySqrt(8))
print("sqrt(16):", mySqrt(16))
```
