## ğŸ” 2. Pull Requests (PRs)
The bread and butter of collaboration. A PR asks the maintainer to merge your branch into the main codebase.

**Process**:
1. Fork repository (if open source) or Create Branch.
2. Push branch to GitHub: `git push origin my-feature`.
3. Open "New Pull Request" on GitHub website.
4. Discuss, Review, Merge.
