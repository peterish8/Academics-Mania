## ğŸ“Œ 1. Frequency Distribution Table
Grouping raw data into classes.

**Components**:
- **Class Interval**: The range (e.g., 10-20).
- **Lower Limit**: Smallest value (10).
- **Upper Limit**: Largest value (20).
- **Class Mark (Midpoint)**: $(Lower + Upper) / 2$.
- **Frequency ($f$)**: Count of items in that class.
- **Cumulative Frequency ($cf$)**: Running total.

### **Example Data**
[5, 12, 15, 22, 28]

| Class | Tally | Frequency ($f$) | Cumulative ($cf$) |
| :---: | :---: | :---: | :---: |
| 0-10 | \| | 1 | 1 |
| 10-20 | \|\| | 2 | 3 |
| 20-30 | \|\| | 2 | 5 |

## ğŸ“ 2. Rules for Tabulation
- Identify Min and Max.
- Choose number of classes ($k$). Rule of thumb: $k = \sqrt{N}$ or Sturges' Rule $k = 1 + 3.322\log(N)$.
- Calculate Class Width: $W = (Max - Min) / k$.
